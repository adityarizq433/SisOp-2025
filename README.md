### 2. Konversi Bilangan Desimal ke Biner
a. **1234₁₀** = **10011010010₂**  
**Cara Pengerjaan:**  
1234 ÷ 2 = 617 sisa 0 <br>
617 ÷ 2 = 308 sisa 1 <br>
308 ÷ 2 = 154 sisa 0 <br>
154 ÷ 2 = 77 sisa 0 <br>
77 ÷ 2 = 38 sisa 1 <br>
38 ÷ 2 = 19 sisa 0 <br>
19 ÷ 2 = 9 sisa 1 <br>
9 ÷ 2 = 4 sisa 1 <br>
4 ÷ 2 = 2 sisa 0 <br>
2 ÷ 2 = 1 sisa 0 <br>
1 ÷ 2 = 0 sisa 1<br>
Hasil: **10011010010₂**

b. **5670₁₀** = **1011000100110₂**  
c. **2321₁₀** = **100100010001₂**

0 × 2⁶ = 0 <br>
0 × 2⁵ = 0 <br>
0 × 2⁴ = 0 <br>
0 × 2³ = 0 <br>
0 × 2² = 0 <br>
0 × 2¹ = 0 <br>
0 × 2⁰ = 0 <br>
Hasil: **Negatif 128**
